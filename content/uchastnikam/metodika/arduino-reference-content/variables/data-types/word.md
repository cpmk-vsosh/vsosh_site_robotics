**Description**

A word can store an unsigned number of at least 16 bits (from 0 to 65535).

**Syntax**

`word var = val;`

**Parameters**

`var`: variable name.\
`val`: the value to assign to that variable.

**Example Code**

``` arduino
word w = 10000;
```
