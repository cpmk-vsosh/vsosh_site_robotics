**Description**

`size_t` is a data type capable of representing the size of any object in bytes. Examples of the use of `size_t` are the return type of `sizeof()` and `Serial.print()`.

**Syntax**

`size_t var = val;`

**Parameters**

`var`: variable name.\
`val`: the value to assign to that variable.
