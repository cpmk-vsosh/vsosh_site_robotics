**Description**

Returns the length of the String, in characters. (Note that this doesn’t include a trailing null character.)

**Syntax**

`myString.length()`

**Parameters**

`myString`: a variable of type `String`.

**Returns**

The length of the String in characters. Data type: `unsigned int`.
